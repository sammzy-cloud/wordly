#  My Dictionary App

A simple dictionary web application built with HTML, CSS, and JavaScript. It allows users to search for English words, view their meanings, pronunciation, examples, synonyms, save words for later, and add words to a favourites list.

## Features

- Search for English words using the Dictionary API
- Display:
- Word
- Definition
- Part of speech
- Pronunciation
- Example sentence
- Synonyms
- Play pronunciation audio
- Save words
- Add words to favourites
- Delete saved and favourite words
- Stores saved and favourite words using Local Storage

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Dictionary API (https://dictionaryapi.dev)

## How to Use

1. Enter an English word in the search bar.
2. Click **Search**.
3. View the word's information.
4. Click **Save** to add the word to your saved words list.
5. Click **Favourite** to add the word to your favourites.
6. Click **Delete** to remove a word from either list.
7. Click the **Audio** button to hear the pronunciation (if available).
